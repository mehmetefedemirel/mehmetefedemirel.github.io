Automated Greenhouse Irrigation System

In Progress

Project Description

This project is designed to automate the irrigation process in a greenhouse using an Arduino Uno. The system utilizes a Real-Time Clock (RTC) module to schedule watering times accurately, a relay module to control the water pump, and is powered by a 9V battery. By automating the irrigation based on pre-set schedules rather than constant manual operation, the system ensures timely watering, reduces water waste, and supports healthier plant growth. It is a cost-effective and energy-efficient solution ideal for small to medium-scale greenhouse operations.

Key Features

  • Automatic Irrigation Control
  • Time Management with RTC Module
  • Low Energy Consumption with 9V Battery
  • Manual Intervention Option
