Arduino Weather Station

Completed

Project Description

In this project, I developed a versatile Weather Monitoring System using Arduino technology. The system captures real-time environmental parameters such as temperature, humidity, atmospheric pressure, and air quality, offering reliable data for environmental research, smart farming, and weather trend analysis.

Key Features

  • Real-time temperature and humidity monitoring
  • Barometric pressure measurement
  • Air quality index calculation
  • Data logging and storage
  • Graphical data visualization

Technical Specifications

Microcontroller

Arduino Uno

Temperature Sensor

DHT22

Pressure Sensor

BMP280

Air Quality Sensor

MQ135

Display

16x2 LCD

Storage

SD Card Module

Key Features

Temperature Monitoring

Accurate temperature readings with ±0.5°C precision

Humidity Tracking

Relative humidity measurement with ±2% accuracy

Pressure Analysis

Barometric pressure monitoring for weather prediction

Air Quality

CO2 and air quality index measurement
